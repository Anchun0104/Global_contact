# 外联教授数据库管理系统

存储国外教授信息并管理与出版社会议的关联合作的 Web 应用。

## 功能

| 功能 | 说明 |
|------|------|
| 教授管理 | 增删改查教授信息（姓名/职称/学校/邮箱/官网/QS排名/研究方向等） |
| 会议管理 | 管理主会信息，支持添加分会场 |
| 分会场分配 | 为分会场分配教授及合作类型，实时统计分配进度 |
| 合作记录 | 全面记录教授与会议的每次合作 |
| 研究方向管理 | 预定义方向列表 + 自定义输入，自动补全 |
| 潜在推荐 | 自动匹配同领域未合作的教授→会议推荐 |
| 仪表盘 | ECharts 图表 + 成员工作量统计 |
| 导入导出 | 支持 JSON / CSV / Excel 三种格式 |
| **多用户系统** | 每人独立账号，管理员管理用户 |
| **操作审计** | 全程记录谁做了什么操作 |
| **工作量统计** | 按用户统计新增教授/会议/合作数量 |

## 部署到 Zeabur（推荐）

1. 在 [zeabur.com](https://zeabur.com) 注册（支持微信登录）
2. 在 GitHub 创建仓库，推送代码：
   ```bash
   git init
   git add .
   git commit -m "init"
   git remote add origin https://github.com/你的用户名/你的仓库名.git
   git push -u origin master
   ```
3. 在 Zeabur 中导入该 GitHub 仓库
4. 在 Zeabur 项目设置中添加环境变量：
   - `SECRET_KEY` — 任意随机字符串，用于 JWT 加密
   - `ADMIN_PASSWORD` — 管理员密码（默认 admin123）
5. 部署完成后访问 Zeabur 分配的域名即可使用

> 本地运行：`pip install -r requirements.txt && python main.py`

## 默认管理员

- 用户名：`admin`
- 密码：`.env` 中 `ADMIN_PASSWORD` 指定的值（默认 `admin123`）

登录后请在「用户管理」页面创建其他用户的账号。

## 数据导入

系统内置批量导入功能，支持 Excel / CSV / JSON。

目录中的模板文件：
- `导入模板_教授.xlsx` — 教授数据导入模板
- `导入模板_会议.xlsx` — 会议数据导入模板
- `导入模板_合作.xlsx` — 合作记录导入模板

在浏览器中打开系统 →「导入导出」页面 → 选择类型和文件 → 点击导入。

支持中文列名，模板文件中有示例数据供参考。
